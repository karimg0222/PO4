Werkgeheugen

Het Werkgeheugen in het Engels: RAM (Random Access Memory) is snelle, tijdelijke opslag waarin gegevens staan die direct worden verwerkt door de processor. In het werkgeheugen staan gegvens van zowel het besturingssysteem, en geopende programma's. Hoe sneller je werkgeheugen en hoe meer opslag je hebt hoe beter je meer dingen tegelijk kan doen (multitasken). Maar hoe meer opslag of snelheid je werkgeheugen heeft, hoe duurder het is. Werkgeheugen is wel ter vergelijking goedkoper dan onderdelen zoals de CPU of GPU (videokaart). Wil je meer weten over het werkgeheugen? Kijk hier dan een kort filmpje

Werkgeheugen begrippen uitgelegd


1: RAM

RAM is de Engelse afkorting voor Random Access Memory, de werkgeheugen van een computer. RAM is tijdelijke opslag waar gegevens van programma's en de processor staan, de RAM biedt snelle toegang tot informatie die de processor gelijk kan gebruiken waardoor processen snel worden uitgevoerd. Als je de computer afsluit worden de gegevens van de RAM gewist. De snelheid van het werkgeheugen wordt gemeten in MHz (megahertz), en laat zien hoe snel data kan worden vervoert per seconde. Een MHz van 3000 betekent dat de RAM data 3000 miljoen keer per seconde kan vervoeren. De type RAM wordt in DDR (Double Data Rate) weergegeven, je hebt DDR1 t/m DDR5. Tegenwoordig heb je in computers alleen werkgeheugen van minstens DDR3, de snehleid van DDR3 is max 1600MHz, van DDR4 max 3200MHz en van DDR5 6400MHz+. RAM komt in sticks van GB (gigabyte) tegenwoordig worden in normale computers sticks van 4 t/m/ 32GB gebruikt, dit kan hoger of lager maar hoe meer GB hoe sneller en hoe meer processen je kan uitvoeren. Grote servers hebben vaak 128+ GB aan RAM vanwege het grote aantal processen dat ze moeten uitvoeren. Lees meer over RAM

2: Cache

Een RAM cache is een deel van het werkgeheugen dat wordt gebruikt om veelgebruikte data van een langzamer opslagapparaat, zoals een harde schijf of SSD, tijdelijk op te slaan. Het doel is om de prestaties van het systeem te verbeteren door deze data sneller toegankelijk te maken voor de CPU en de benodigde toegangstijd te verkorten. 3: ROM

ROM (Read Only Memory) is een type geheugen dat dat permanente gegevens opslaat, zoals basissoftware die nodig is om te computer te starten. Deze gegevens blijven op de computer staan, en zijn niet bedoelt om veranderd worden maar alleen om gelezen te worden door de computer. Lees meer over de ROM
