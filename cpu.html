Central Processing Unit

De CPU (afkorting: Central Processing Unit) is het brein van de computer, en leest instructies, en voert miljarden berekeningen per seconde uit en geeft de resultaten door aan de rest van je PC. Hij ziet eruit als een chip op je computer haalt instructies op uit het geheugen, decodeert ze en voert ze uit. Een CPU heeft vaak meerdere kernen, waarbij elke kern een onafhankelijke eenheid is die instructies kan uitvoeren. Meer kernen betekent dat de CPU meer taken tegelijkertijd kan uitvoeren, en hoe meer kernen hoe duurder de CPU. Wil je meer weten over hoe een CPU werkt? Kijk hier dan een kort filmpje

CPU begrippen uitgelegd


1: ALU

De ALU (Arithmetic logic unit) is het centrale onderdeel van de processor dat rekenkundige en logische berekeningen uitvoert. Een ALU is een verzameling van poorten die in nanoseconden in bits pure aan of uit binaire code hele snelle berekeningen uitvoeren. Lees meer over de ALU

2: Registers

Registergeheugen is intern geheugen dat gebruikt wordt in processors om waarden op te slaan waar de processor direct toegang toe moet hebben om zijn werk te kunnen voltooien. Een processor haalt voortdurent drie stappen uit: haal de volgende instructie op, decodeer de instructie en voer de instructie uit. Lees meer over Registers

3: Control Unit

Een besturingseenheid of control unit is een onderdeel van de CPU van een computer die de meeste bewerkingen van de processor aanstuurt en co√∂rdineert door instructies uit het geheugen op te halen, te decoderen en uit te voeren. De control unit haalt een instructie op, vertaalt hem en voert hem uit. Telkens opnieuw. Lees meer over de Control Unit

4: Instructie Cyclus

De instructie cyclus is een cyclus dat de CPU volgt om processen te verwerken, en volgt 3 fases: de ophaalfase, de decodeerfase en de uitvoeringsfase zoals eerder vermeld bij de registers. Lees meer over de instructie cyclus van een CPU

5: Machinetaal

Machinetaal is de basistaal van een computer, die direct door de CPU wordt uitgevoerd en bestaat uit binaire getallen die instructies en gegevens voorstelle, elke type processor heeft zijn eigen unieke machinetaal. Menselijke programmeertalen (Python, C, Java, etc.) worden vertaald naar machinetaal door compilers en interpreters. Lees meer over de machinetaal van een CPU

6: Assembleertaal

Assemblertaal is een symbolische weergave van machinetaal, elke assembly instructie komt overeen met een specifieke machine instructie die de processor van de computer rechtstreeks kan uitvoeren. In plaats van binaire getallen gebruikt het codes die makkelijker leesbaar zijn voor mensen, om direct instructies te geven aan de processor van een computer en geeft programmeurs directe controle over de hardware. Lees meer over assemblertaal

7: Programmeertalen

Een programmeertaal is een speciale taal met een eigen grammatica (syntax) om instructies aan een computer te geven voor het uitvoeren van taken. Verschillende talen worden gebruikt voor verschillende software of programma's, C# die vaak voor games wordt gebruikt of Python voor data analyse.
